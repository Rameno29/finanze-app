# Istruzioni permanenti del progetto

- Leggere integralmente `README_FIRST.md` prima di modificare il progetto.
- Dopo ogni modifica significativa a codice, configurazione, sicurezza, test, database o
  infrastruttura, aggiornare automaticamente `README_FIRST.md` nello stesso intervento.
- Aggiornare la data “Ultimo aggiornamento” e le sezioni pertinenti prima della consegna finale.
- Non aggiornare il documento per sole letture, verifiche o diagnosi che non cambiano file o servizi.
- Non inserire mai nel README segreti, token, password o dati personali non già pubblici.
